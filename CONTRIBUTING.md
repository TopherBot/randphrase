# Contributing to randphrase

Thank you for considering contributing! 🎉

## How to get started

1. **Fork** the repository and clone your fork.
2. Create a new branch for your feature or bug‑fix:
   ```bash
   git checkout -b my‑awesome‑feature
   ```
3. Make your changes. Keep the code minimal and well‑documented.
4. Add or update tests in the `tests/` directory.
5. Run the test suite locally:
   ```bash
   python -m pytest -q
   ```
6. Commit with a clear message and push to your fork.
7. Open a **Pull Request** against the `main` branch.

## Code style

- Follow PEP 8 (use `black` and `flake8` for formatting & linting).
- Keep import statements tidy and avoid unnecessary dependencies.

## Testing

All new code must have at least one unit test. We use **pytest** for testing.

## Issues & Feature Requests

Feel free to open an issue. Tag it with `good first issue` if it is suitable for newcomers.
